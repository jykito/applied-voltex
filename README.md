# Applied Voltex

Applied Voltex is a standalone Applied Energistics 2 addon for Minecraft 1.20.1 (Forge). It adds an Instant Assembler (a grid-connected pattern provider that crafts instantly into network storage), an Avaritia Assembler for 9x9 processing patterns, and an Avaritia Encoder that turns Avaritia table recipes into AE2 processing patterns.

Downloads and the full description are on CurseForge:
https://www.curseforge.com/minecraft/mc-mods/appliedvoltex

## Dependencies

- Minecraft 1.20.1, Forge 47.x
- Applied Energistics 2 (required)
- Re:Avaritia (optional, enables the Avaritia encoding features)

## Building

Standard ForgeGradle. From the project root:

    ./gradlew build

The built jar is placed in `build/libs/`.
